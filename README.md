## Setup
```bash
conda create --name uf python=3.7
conda activate uf
pip install numpy scipy tqdm ase pandas scikit-learn
git clone https://github.com/sxie22/fast-linear-qmml
cd fast-linear-qmml
python setup.py sdist
pip install -e .
```
